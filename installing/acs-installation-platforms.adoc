[id="install-platforms-methods"]
== Installation methods for different platforms

You can perform different types of installations on different platforms.

[NOTE]
====
Not all installation options are supported for all platforms, as shown in the following tables. Red Hat recommends that you do not use the `roxctl` install method unless you have a specific installation need that requires using this method.
====

.Self-managed platforms
|===
|Platform | Supported installation methods

|Red Hat OpenShift Container Platform (OCP) 4.x a|
* xref:../installing/install-ocp-operator.adoc#install-ocp-operator[Operator] (recommended)
* xref:../installing/installing_helm/install-helm-quick.adoc#acs-install-helm-quick[Helm]
* xref:../installing/install-quick-roxctl.adoc#install-quick-roxctl[`roxctl`]

|Red Hat OpenShift Container Platform (OCP) 3.11.z a|
* xref:../installing/installing_helm/install-helm-quick.adoc#acs-install-helm-quick[Helm] (recommended)
* xref:../installing/install-quick-roxctl.adoc#install-quick-roxctl[`roxctl`]

|Red Hat OpenShift Kubernetes Engine (OKE) 4.x a|
* xref:../installing/install-ocp-operator.adoc#install-ocp-operator[Operator] (recommended)
* xref:../installing/installing_helm/install-helm-quick.adoc#acs-install-helm-quick[Helm]
* xref:../installing/install-quick-roxctl.adoc#install-quick-roxctl[`roxctl`]

|===
